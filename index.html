Introduction to CSS

Welcome to CSS Styling

This document demonstrates various CSS techniques and selectors to style an HTML page. CSS (Cascading Style Sheets) is the language we use to style web documents, controlling the visual presentation of HTML elements.

Different Types of Selectors

CSS uses selectors to target HTML elements. Here are some common types:

  • Element Selector - Targets HTML elements by their tag name
  • Class Selector - Targets elements with a specific class attribute
  • ID Selector - Targets a specific element with a unique ID
  • Descendant Selector - Targets elements that are descendants of another element
  • Pseudo-class Selector - Targets elements in a specific state

Typography and Spacing

CSS allows us to control text appearance and spacing between elements:

  • Font families, sizes, and weights
  • Text alignment and decoration
  • Line heights and letter spacing
  • Margins (space outside elements)
  • Padding (space inside elements)
  • Borders around elements
